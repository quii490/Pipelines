# Transposable Element 分析

| 问题 | 合适层级 |
|---|---|
| 哪些 TE family/subfamily 改变？ | family/subfamily counts |
| 哪些具体 TE locus 改变？ | locus-level allocation |
| 某蛋白是否富集在 TE？ | peak overlap、TE signal/profile |
| TE 区域整体开放性如何？ | ATAC TE annotation/profile |

Multi-mapping 处理、annotation 版本和计量层级会显著改变结论。报告中至少记录 genome build、RepeatMasker 版本、family/subfamily/locus 层级、multi-mapper 策略与 normalization。
