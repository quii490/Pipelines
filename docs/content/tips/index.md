# 实用技巧

这一栏不重复 Quick Start 或 pipeline 教程，而是回答日常工作中经常卡住、但不属于某一条 pipeline 的问题。

| 场景 | 从这里开始 |
|---|---|
| 第一次连接 Linux/HPC | [登录和使用计算集群](cluster-login.md) |
| 不知道怎样申请 CPU/内存、查看或取消作业 | [Slurm、后台任务与资源](cluster-jobs.md) |
| 传文件、找日志、保持远程会话 | [Shell、日志与文件传输](shell-productivity.md) |
| 想让 AI 解释报错、写脚本或审阅结果 | [使用 AI 辅助生信工作](ai-assisted-work.md) |
| 想让一次分析以后还能复现和交付 | [分析记录与交付习惯](analysis-habits.md) |

这里的命令使用通用示例：`user@login.example.org`、`/path/to/...` 和 `slurm`。实际主机名、账户、分区、存储路径和安全政策应从实验室内部文档获取，不进入公开仓库。
