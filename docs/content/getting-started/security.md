# 数据安全与公开内容

| 状态 | 维护人 | 最后审查 | 适用版本 |
|---|---|---|---|
| Active | Lab pipeline maintainers | 2026-07-15 | `main` |

## 可以公开

- 通用运行方法、参数解释、公开参考资源名称。
- 合成或已明确授权的最小测试数据。
- 不含身份信息的示意图和预期输出结构。

## 不得公开

- 密码、Token、SSH key、内网地址和私有资源路径。
- 患者信息、真实样本编号、未发表数据或结果。
- FASTQ、BAM、bigWig、分析结果目录和完整运行日志。

提交前运行：

```bash
bash scripts/validate_public_repo.sh
```

如果某项信息对内部运行必需，应放入不跟踪的本地配置，并在仓库中提供脱敏的 `.example` 模板。
